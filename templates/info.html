About the platform

This platform allows to automatically trace the contour of a dorsal fin. The result is obtained using a neural network which extracts the features of the fin outline and automatically produces the contour of the fin. For our purpose we used a conditional generative adversarial network (cGAN) called pix2pix that learns a mapping from input images to output images, realizing an image translation. For our purpose we used the discriminator only to train the model whereas we use the generator to create the outline. The generator has a U-Net based architecture. The U-Net architecture was developed for image segmentation purposes. We applied some modification to the standard architecture to make the network suitable for our goal.
